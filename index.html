Desarrollo web

El desarrollo web es un proceso complejo que implica la creación
de productos digitales en línea. También es un término que define
la creación de sitios web para Internet o una intranet. Para conse-
guirlo se hace uso de tecnologías de software del lado del servidor
y del cliente que involucran una combinación de procesos de base
de datos con el uso de un navegador web a fin de realizar determi-
nadas tareas o mostrar información.

Marketing Digital

El mercadeo digital o marketing digital es el componente de la
mercadotecnia que utiliza internet y tecnologías digitales en línea,
como computadoras de escritorio, teléfonos móviles y otras
plataformas y medios digitales para promover productos y servicios.

Desarrollo de Apps

El desarrollo de aplicaciones es el proceso de crear programas
informáticos para realizar o apoyar una función específica de un
negocio. Este proceso puede incluir varias fases, como la
planificación, el diseño, la creación, las pruebas y la implementación.
